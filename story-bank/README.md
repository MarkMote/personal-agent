# Story Bank

Pre-written stories for interviews. Pull from these when asked behavioral questions.

## How to Use
1. Fill in `stories.md` with your 5-7 go-to stories
2. Review before each interview
3. Tag which stories work for which company types

## Story Format

Each story should follow STAR format:
- **Situation:** Context (1-2 sentences)
- **Task:** What you needed to do
- **Action:** What you specifically did
- **Result:** Quantified outcome

Keep each story under 2 minutes when spoken aloud.

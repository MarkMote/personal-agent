# 2026-02-24 | DRAFT | Archer Case Study - Title & Abstract

**Title:** Safe Autonomy with Runtime Assurance

**Abstract:**

Runtime assurance (RTA) provides safety guarantees for autonomous systems by monitoring behavior in real time and overriding unsafe commands before constraints are violated. This talk introduces the core theory behind RTA then demonstrates these ideas through three spacecraft applications. First, implicit safe set methods applied to the proximity operations problem (ARPOD), enforcing collision avoidance and approach corridor constraints during rendezvous. Second, an extension to nondeterministic systems for attitude control, using reachable set overapproximation to enforce line-of-sight keep-out constraints despite unknown disturbances. Third, collision-inclusive trajectory optimization, which expands the feasible set for motion planning by modeling collisions as constraints rather than treating them as failures.
